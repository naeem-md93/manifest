1. Hello World Generator
   - ایجاد یک پروژه پایه با استفاده از ReactJS و FastAPI.
   - تنظیم ساختار دایرکتوری و فایل‌های ضروری برای فرانت اند و بک اند.

2. ایجاد چت بات AI
   - ساخت یک چت بات با استفاده از LangChain و LangGraph.
   - طراحی مدل تصمیم‌گیری برای تشخیص تصمیم‌های زندگی کاربر.

3. ذخیره برنامه در فایل JSON
   - طراحی ساختار داده برای ذخیره برنامه‌های روزانه، هفتگی و سالانه.
   - ایجاد ورودی و خروجی برای انتقال داده‌های کاربر.

4. گزارش هفتگی
   - طراحی مکانیسم گزارش هفتگی که هر شب از کاربر می‌خواهد تسک‌های روز را گزارش دهد.
   - ایجاد فرم گزارش با موارد انجام شده، چالش‌ها و پیشرفت.

5. به‌روزرسانی برنامه
   - ایجاد مکانیسمی برای اصلاح برنامه با استفاده از گزارش هفتگی.
   - تطبیق برنامه با پیشرفت کاربر و ایجاد برنامه‌های شخصی‌سازی شده.

6. حفظ حریم خصوصی و امنیت داده‌ها
   - ایجاد مکانیسم‌های امنیتی برای محافظت از داده‌های کاربر.
   - استفاده از روش‌های مخفی‌سازی و دسترسی محدود.

7. تضمین دقت و سبک بین‌المللی
   - تنظیم پارامترهای مدل AI برای اطمینان از دقت و سبک بین‌المللی.

8. پشتیبانی از تغییرات کاربر
   - طراحی مکانیسمی برای به‌روزرسانی برنامه با تغییرات کاربر در طول زمان.
